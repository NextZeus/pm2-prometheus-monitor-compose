# Prometheus Docker-Compose

### Command

```

#启动容器：
docker-compose -f /data/work/prometheus-monitor/config/docker-compose-monitor.yml up -d
#删除容器：
docker-compose -f /data/work/prometheus-monitor/config/docker-compose-monitor.yml down
#重启容器：
docker restart id

```
